Discord Forum Ping Bot
----------------------

This bot automatically pings a role whenever a new forum post (thread) is created.

FILES INCLUDED:
- index.js
- package.json
- keepalive.js

HOW TO USE IN REPLIT:
1. Upload these files.
2. Go to Secrets and add:
   BOT_TOKEN = your Discord bot token
   ROLE_ID = role ID to ping
3. Run:
   npm install
   npm start
